# hamacmonitor

This is an operating level system dashboard with different stats.

Note: _Built for macos, may not work on linux or windows_

## Installation

The built files are inside of the Releases tab, However, if you want to build if from scratch run these commands

### Dev Dependencies (Building the code)

- GO

```bash
  git clone https://github.com/myshy13/hamacmonitor.git
  cd hamacmonitor
  go build -o hamacmonitor
```
